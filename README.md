# maze-game

Empty project.

## Building and running on localhost

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build-prod
```

To create a development build:

```sh
npm run build-dev
```

Start development server:

```sh
npm run serve
```

## Running

```sh
node dist/bundle.js
```
